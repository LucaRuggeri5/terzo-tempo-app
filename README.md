# ⚽ TERZO TEMPO - Gestione Partite Calcio Amatoriale

**TERZO TEMPO** è una web app moderna progettata per semplificare l'organizzazione e il monitoraggio delle statistiche per i gruppi di amici che giocano a calcio. Dalla creazione delle formazioni al tracciamento dei marcatori, tutto in un unico posto.

---

## 🚀 Funzionalità Principali

- **Dashboard Statistiche:** Visualizzazione rapida delle performance generali.
- **Classifica Marcatori:** Tracciamento automatico dei goal per ogni giocatore.
- **Gestione Partite:** Formazione delle squadre (Nera vs Bianca) con sistema di convocazioni rapido.
- **Storico Match:** Registro completo di tutte le partite giocate, con punteggi e marcatori.
- **Responsive Design:** Ottimizzata per l'uso su smartphone direttamente a bordo campo.

## 🛠️ Tech Stack

- **Frontend:** [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Styling:** CSS3 (Custom Modules) con approccio Mobile-First.
- **Database:** [Supabase](https://supabase.com/) (PostgreSQL) - *In fase di implementazione*
- **Hosting:** [Vercel](https://vercel.com/) - *In fase di implementazione*
